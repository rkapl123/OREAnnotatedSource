61 class CdsHelper : public RelativeDateDefaultProbabilityHelper {
62 public:
63  /*! Constructor
64  @param quote The helper's market quote.
65  @param tenor CDS tenor.
66  @param settlementDays The number of days from evaluation date to the start of the protection period. Prior to
67  the CDS Big Bang in 2009, this was typically 1 calendar day. After the CDS Big Bang,
68  this is typically 0 calendar days i.e. protection starts immediately.
69  @param calendar CDS calendar. Typically weekends only for standard non-JPY CDS and TYO for JPY.
70  @param frequency Coupon frequency. Typically 3 months for standard CDS.
71  @param paymentConvention The convention applied to coupons schedules and settlement dates.
72  @param rule The date generation rule for generating the CDS schedule. Typically, for CDS prior to the Big
73  Bang, \c OldCDS should be used. After the Big Bang, \c CDS was typical and since 2015 \c CDS2015
74  is standard.
75  @param dayCounter The day counter for CDS fee leg coupons. Typically it is Actual/360, excluding accrual end,
76  for all but the final coupon period with Actual/360, including accrual end, for the final
77  coupon. The \p lastPeriodDayCounter below allows for this distinction.
78  @param recoveryRate The recovery rate of the underlying reference entity.
79  @param discountCurve A handle to the relevant discount curve.
80  @param startDate Used to specify an explicit start date for the CDS schedule and the date from which the
81  CDS maturity is calculated via the \p tenor. Useful for off-the-run index schedules.
82  @param settlesAccrual Set to \c true if accrued fee is paid on the occurence of a credit event and set to
83  \c false if it is not. Typically this is \c true.
84  @param protectionPaymentTime Time at which protection payments are made when there is a credit event.
85  @param lastPeriodDayCounter The day counter for the last fee leg coupon. See comment on \p dayCounter.
86  */
87  CdsHelper(const Handle<Quote>& quote, const Period& tenor, Integer settlementDays, const Calendar& calendar,
