Detailed Description

multi section default ts

the instantaneous hazard rate is defined by the ith source curve for dates before the ith switch date and after the (i-1)th switch date; all source curves must be conssitently floating or fixed and have the same reference date always; the day counter of all source curves should coincide with the dc of this curve
