familyName may be e.g. ECB settlementDays determine the spot date of the currency pair source is the asset or foreign currency target is the numeraire or domestic currency fixingCalendar is the calendar defining good days for the pair this class uses the exchange rate manager to retrieve spot values

if inverseIndex is true, the returned fixing values are flipped, but all inspectors

  • sourceCurremcy(), targetCurrency()
  • sourceCurve(), targetCurve()
  • fxQuote() will still return results in terms of the original pair.
