Detailed Description

Class that converts a supplied SwaptionVolatilityStructure to one of another type with possibly different shifts.

Warning:
the converted SwaptionVolatilityStructure object has a fixed reference date eqaul to asof and fixed market data regardless of the type of reference date and market data of the original SwaptionVolatilityStructure that is passed in
