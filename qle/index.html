Introduction

The Open Source Risk Project aims at establishing a framework for transparent pricing and risk analysis that can be adapted for production purposes in financial institutions and used as a benchmarking and validation tool to test in-house and vendor solutions applied in the industry.

OpenSourceRiskEngine aka ORE (http://opensourcerisk.org) is the software project implementing this framework. ORE comprises three libraries written in C++. It is based on QuantLib (http://quantlib.org), the free/open source library for quantitative finance.
