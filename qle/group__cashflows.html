file  nonstandardinflationcouponpricer.hpp
 pricer for the generalized (nonstandard) yoy coupon the payoff of the coupon is: N * (alpha * I_t/I_s + beta) N * (alpha * (I_t/I_s - 1) + beta) with arbitrary s < t.
