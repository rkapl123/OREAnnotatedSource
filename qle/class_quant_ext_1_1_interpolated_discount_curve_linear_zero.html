Detailed Description

InterpolatedDiscountCurveLinearZero as in QuantLib, but with floating discount quotes and floating reference date.

InterpolatedDiscountCurveLinearZero as in QuantLib, but with floating discount quotes and floating reference date, reference date is always the global evaluation date, i.e. settlement days are zero and calendar is NullCalendar()
