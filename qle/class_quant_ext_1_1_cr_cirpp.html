Detailed Description

Cox-Ingersoll-Ross ++ credit model class.

This class implements the Cox-Ingersoll-Ross model defined by

\[ \lambda(t) = y(t) + \psi (t) \\ dy(t) = a(\theta - y(t)) dt + \sigma \, \sqrt{y(t)} \, dW \]
