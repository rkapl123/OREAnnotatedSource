182 // Test yield curve bootstrap from overnight index futures where the first future in the list of instruments may be
183 // expired. We use the March 2020 SOFR future contract whose last trade date is 16 Jun 2020 with settlement date 17
184 // Jun 2020. A number of cases are tested:
185 // 1. Valuation date is 9 Jun 2020. March 2020 SOFR future should be included in bootstrap fine.
186 // 2. Valuation date is 16 Jun 2020. March 2020 SOFR future should be included in bootstrap. The final SOFR fixing
187 // i.e. the fixing for 16 Jun 2020 will not be known on 16 Jun 2020.
188 // 3. Valuation date is 17 Jun 2020. March 2020 SOFR future should be excluded from the bootstrap.
189 // 4. Valuation date is 23 Jun 2020. March 2020 SOFR future should be excluded from the bootstrap.
190

192  Date date;
193  string desc;
194 };

197  {Date(16, Jun, 2020), "on_ltd"},
198  {Date(17, Jun, 2020), "on_settlement"},
199  {Date(23, Jun, 2020), "after_ltd"}};

201 // Test yield curve bootstrap from money market futures where the first future in the list of instruments has an ibor
202 // start date that is before, on and after the valuation date. We use the August 2020 Eurodollar future contract whose
203 // last trade date is 17 Aug 2020 with an underlying ibor start date of 19 Aug 2020. Note that the USD-LIBOR-3M fixing
204 // is known on 17 Aug 2020 and the future expires on this date with the associated final settlement price. A number of
205 // cases are tested:
206 // 1. Valuation date is 18 Aug 2020. August 2020 Eurodollar future should be included in bootstrap.
