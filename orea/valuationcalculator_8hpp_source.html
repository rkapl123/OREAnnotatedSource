94 //! CashflowCalculator
95 /*! Calculates the cashflow, converted to base ccy, from t to t+1, this interval is defined by the provided dategrid
96  * The interval is (t, t+1], i.e. we exclude todays flows and include flows that fall exactly on t+1.
97  * For t0 we do nothing (and so the cube will have a 0 value)
98  */
99 class CashflowCalculator : public ValuationCalculator {
100 public:
101  //! Constructor takes the base ccy, date grid and index of cube to write to.

105

109 
110  virtual void calculateT0(const boost::shared_ptr<Trade>& trade, Size tradeIndex,
