48 //! Sensitivity Analysis
49 /*!
50  This class wraps functionality to perform a sensitivity analysis for a given portfolio.
51  It comprises
52  - building the "simulation" market to which sensitivity scenarios are applied
53  - building the portfolio linked to this simulation market
54  - generating sensitivity scenarios
55  - running the scenario "engine" to apply these and compute the NPV impacts of all required shifts
56  - compile first and second order sensitivities for all factors and all trades
57  - fill result structures that can be queried
58 
59  \ingroup simulation
60 */
61
