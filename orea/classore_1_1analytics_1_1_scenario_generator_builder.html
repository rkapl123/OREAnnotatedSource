Detailed Description

Build a ScenarioGenerator.

ScenarioGeneratorBuilder builds a scenario generator based on the settings provided via the ScenarioGeneratorData object

  • state process
  • simulation date grid
  • multipath generator
  • scenario factory
  • fixing method
