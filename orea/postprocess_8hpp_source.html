48 //! Exposure Aggregation and XVA Calculation
49 /*!
50  This class aggregates NPV cube data, computes exposure statistics
51  and various XVAs, all at trade and netting set level:
52 
53  1) Exposures
54  - Expected Positive Exposure, EPE: E[max(NPV(t),0) / N(t)]
55  - Expected Negative Exposure, ENE: E[max(-NPV(t),0) / N(t)]
56  - Basel Expected Exposure, EE_B: EPE(t)/P(t)
57  - Basel Expected Positive Exposure, EPE_B
58  - Basel Effective Expected Exposure, EEE_B: max( EEE_B(t-1), EE_B(t))
59  - Basel Effective Expected Positive Exposure, EEPE_B
60  - Potential Future Exposure, PFE: q-Quantile of the distribution of
61 
62  2) Dynamic Initial Margin via regression
63 
64  3) XVAs:
65  - Credit Value Adjustment, CVA
66  - Debit Value Adjustment, DVA
67  - Funding Value Adjustment, FVA
68  - Collateral Value Adjustment, COLVA
69  - Margin Value Adjustment, MVA
70 
71  4) Allocation from netting set to trade level such that allocated contributions
72  add up to the netting set
73  - CVA and DVA
74  - EPE and ENE
75 
76  All analytics are precomputed when the class constructor is called.
77  A number of inspectors described below then return the individual analytics results.
78 
79  Note:
80  - exposures are discounted at the numeraire N(t) used in the
81  Monte Carlo simulation which produces the NPV cube.
82  - NPVs take collateral into account, depending on CSA settings
83 
84  \ingroup analytics
85 
86  \todo Introduce enumeration for TradeAction type and owner
87  \todo Interpolation for DIM(t-MPOR) when the simulation grid spacing is different from MPOR
88  \todo Revise alternatives to the RelativeXVA exposure and XVA allocation method
89  \todo Add trade-level MVA
90  \todo Take the spread received on posted initial margin into account in MVA calculation
91 */
