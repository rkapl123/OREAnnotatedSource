56 //! Exposure Aggregation and XVA Calculation
57 /*!
58  This class aggregates NPV cube data, computes exposure statistics
59  and various XVAs, all at trade and netting set level:
60 
61  1) Exposures
62  - Expected Positive Exposure, EPE: E[max(NPV(t),0) / N(t)]
63  - Expected Negative Exposure, ENE: E[max(-NPV(t),0) / N(t)]
64  - Basel Expected Exposure, EE_B: EPE(t)/P(t)
65  - Basel Expected Positive Exposure, EPE_B
66  - Basel Effective Expected Exposure, EEE_B: max( EEE_B(t-1), EE_B(t))
67  - Basel Effective Expected Positive Exposure, EEPE_B
68  - Potential Future Exposure, PFE: q-Quantile of the distribution of
69 
70  2) Dynamic Initial Margin via regression
71 
72  3) XVAs:
73  - Credit Value Adjustment, CVA
74  - Debit Value Adjustment, DVA
75  - Funding Value Adjustment, FVA
76  - Collateral Value Adjustment, COLVA
77  - Margin Value Adjustment, MVA
78 
79  4) Allocation from netting set to trade level such that allocated contributions
80  add up to the netting set
81  - CVA and DVA
82  - EPE and ENE
83 
84  All analytics are precomputed when the class constructor is called.
85  A number of inspectors described below then return the individual analytics results.
86 
87  Note:
88  - exposures are discounted at the numeraire N(t) used in the
89  Monte Carlo simulation which produces the NPV cube.
90  - NPVs take collateral into account, depending on CSA settings
91 
92  \ingroup analytics
93 
94  \todo Introduce enumeration for TradeAction type and owner
95  \todo Interpolation for DIM(t-MPOR) when the simulation grid spacing is different from MPOR
96  \todo Revise alternatives to the RelativeXVA exposure and XVA allocation method
97  \todo Add trade-level MVA
98  \todo Take the spread received on posted initial margin into account in MVA calculation
99 */
