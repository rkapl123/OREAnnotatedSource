164 /*! This calls the private method getReportWriterImpl() which returns the
165 actual ReportWriter implementation. The private method is virtual and

167 instance. This method is not virtual and can be hidden in derived
168 classes by a method with the same name. This method can then return a
169 shared pointer to the derived ReportWriter class.
170
171 For example, we can have the following in a derived class:
172 \code{.cpp}
173 class MyApp : public OREApp {
174 // stuff
175 protected:
176 boost::shared_ptr<MyReportWriter> getReportWriter() {
177 return boost::shared_ptr<MyReportWriter>(getReportWriterImpl());
178 }
179 private:
180 virtual MyReportWriter* getReportWriterImpl() const {
181 return new MyReportWriter();
182 }
183 };
184 \endcode
185 where we have our own special report writer class MyReportWriter that
186 derives from ReportWriter or any class in its hierarchy.
187 */
