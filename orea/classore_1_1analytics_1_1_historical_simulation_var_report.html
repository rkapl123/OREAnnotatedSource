Detailed Description

HistoricalSimulation VaR Calculator.

This class takes sensitivity data and a covariance matrix as an input and computes a Historical Simulation value at risk. The output can be broken down by portfolios, risk classes (IR, FX, EQ, ...) and risk types (delta-gamma, vega, ...).
