47 //! Sensitivity Analysis
48 /*!
49  This class wraps functionality to perform a sensitivity analysis for a given portfolio.
50  It comprises
51  - building the "simulation" market to which sensitivity scenarios are applied
52  - building the portfolio linked to this simulation market
53  - generating sensitivity scenarios
54  - running the scenario "engine" to apply these and compute the NPV impacts of all required shifts
55  - compile first and second order sensitivities for all factors and all trades
56  - fill result structures that can be queried
57 
58  \ingroup simulation
59 */
60
