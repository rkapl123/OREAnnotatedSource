Detailed Description

Sensitivity Analysis.

This class wraps functionality to perform a sensitivity analysis for a given portfolio. It is derived from the ORE SensitivityAnalysis class, and simply overrides one virtual functionality It uses a more space-efficient scenario factory so as to mitigate memory bloat
