109//! Simulation Market updated with discrete scenarios
110/*! If useSpreadedTermStructures is true, spreaded term structures over the initMarket for supported risk factors will
111 be generated. This is used by the SensitivityScenarioGenerator.
112
113 If cacheSimData is true, the scenario application is optimised. This requires that all scenarios are SimpleScenario
114 instances with identical key structure in their data.
115
116 If allowPartialScenarios is true, the check that all simData_ is touched by a scenario is disabled.
117 */
