◆ configureExtrapolation()

void configureExtrapolation ( const std::string &  extrapolation)
private

Set the values of extrapolate_ and flatExtrapolation_ based on the value of extrapolation. The extrapolation string can take the values "Linear", "Flat" or "None".

  • "Linear" is for backwards compatibility and means extrapolation is on and flat extrapolation is off
  • "Flat" means extrapolation is on and it is flat
  • "None" means extrapolation is off
