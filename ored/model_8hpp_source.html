101 /* eval index at (past or future) obsdate:
102 - if fwddate != null, fwddate > obsdate is required. A check must be implemented that the obsdate allows for
103 the index projection. For non-inflation indices this check is simply obsdate >= refdate. For zero inflation
104 indices the is is obsdate >= basedate where the basedate is the one from the zero inflation ts asociated
105 to the index.
106 - if a historical fixing is required and missing, the behaviour depends on returnMissingFixingAsNull, if this
107 flag is true, an exception is thrown, if false, an RandomVariable rv with rv.initialised() = false is returned
108 - for non-inflation indices, if ignoreTodaysFixing is true, always return the market spot for obsdate =
109 referencedate, even if a historical fixing is available; for inflation indices, ignore this flag
110 */
111 virtual RandomVariable eval(const std::string& index, const Date& obsdate, const Date& fwddate,

113 const bool ignoreTodaysFixing = false) const = 0;
