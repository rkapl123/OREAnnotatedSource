51 /*! Calibration instrument factory class
52 
53  This class is a repository of functions that can build instances of \c CalibrationInstrument. The functions are
54  keyed on the calibration instrument type that they can build. An instance of this factory class can be asked to
55  build a particular instance of the CalibrationInstrument class via a call to
56  <code>build(const std::string& instrumentType)</code> with the correct calibration instrument type,
57  \c instrumentType. For example, a call to <code>build("CpiCapFloor")</code> should return a \c CpiCapFloor
58  instance if the CpiCapFloor calibration instrument building function has been added to the factory.
59 
60  It is up to each class derived from \c CalibrationInstrument to register itself with the
61  \c CalibrationInstrumentFactory via the \c CalibrationInstrumentRegister class below. All registration does is add
62  a function that can build an instance of that class to the factory and store it against its calibration instrument
63  type key.
64 
65  \ingroup models
66 */
67 class CalibrationInstrumentFactory
68  : public QuantLib::Singleton<CalibrationInstrumentFactory, std::integral_constant<bool, true>> {

97 /*! Calibration instrument registration class
98 
99  This class is used in any class derived from \c CalibrationInstrument to register itself with the
100  \c CalibrationInstrumentFactory so that it can be built via a call to
101  <code>CalibrationInstrumentFactory::instance().build(const std::string& instrumentType)</code>.
102 
103  As a concrete example, a \c CpiCapFloor class derived from \c CalibrationInstrument should have the following form
104  in order to register it with the \c CalibrationInstrumentFactory:
105 
106  In cpicapfloor.hpp
107  \code{.cpp}
108  class CpiCapFloor : public CalibrationInstrument {
109  public:
110  private:
111  static CalibrationInstrumentRegister<CpiCapFloor> reg_;
112  }
113  \endcode
114 
115  In cpicapfloor.cpp
116  \code{.cpp}
117  CalibrationInstrumentRegister<CpiCapFloor> CpiCapFloor::reg_("CpiCapFloor");
118  \endcode
119 
120  \ingroup models
121 */
122 template <class T> struct CalibrationInstrumentRegister {
123 public:
124  CalibrationInstrumentRegister(const std::string& instrumentType) {
125  CalibrationInstrumentFactory::instance().addBuilder(
