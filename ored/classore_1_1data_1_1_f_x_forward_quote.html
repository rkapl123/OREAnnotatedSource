Detailed Description

Foreign exchange rate data class.

This class holds single market points of type

  • FX_FWD Specific data comprise
  • unit currency
  • currency
  • term
  • conversion factor

The quote is expected in "forward points" = (FXFwd - FXSpot) / conversionFactor
