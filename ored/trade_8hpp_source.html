72  /*! Return the fixings that will be requested in order to price this Trade given the \p settlementDate.
73 
74 
75 
76  If the \p settlementDate is not provided, the current evaluation date is taken as the settlement date.
77  If a Trade does not have any fixings, this method will return an empty map.

80  \warning This method will return an empty map if the Trade has not been built.
81  */
