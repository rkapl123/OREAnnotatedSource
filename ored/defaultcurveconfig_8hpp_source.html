140  /*! Indicates if the reference entity's default status should be implied from the market data. If \c true, this
141  behaviour is active and if \c false it is not. If not explicitly set, it is assumed to be \c false.
142 
143  When a default credit event has been determined for an entity, certain market data providers continue to
144  supply a recovery rate from the credit event determination date up to the credit event auction settlement
145  date. In this period, no CDS spreads or upfront prices are provided.
146 
147  When this flag is \c true, we assume an entity is in default if we find a recovery rate in the market but no
148  CDS spreads or upfront prices. In this case, we build a survival probability curve with a value of 0.0
149  tomorrow. This will give some approximation to the correct price for CDS and index CDS in these cases.
150 
151  When this flag is \c false, we make no such assumption and the default curve building fails.
152  */
153  boost::optional<bool> implyDefaultFromMarket_;
154 
155  /*! If this flag is set to true, the checks for negative hazard rates are disabled when building a curve of type
156  - HazardRate
157  - Benchmark
158  */
159  bool allowNegativeRates_;
